# fs

> **对文件和文件夹进行读写和增删改查**

```js
const fs = require('fs')
```

- **参数**
  - **path<string>: 路径**
  - **option<Object>**
    - **bigInt<boolean>：返回的对象数值是否应为bigInt，默认 false**
  - **callback**
    - **err：错误信息**
    - **state：文件信息**

# 1. 获取文件信息

- **方式一：同步方式**

```js
const fs = require('fs')
fs.statSync('./test.txt')
fs.statSync('./test2')
```

- **方式二：异步方式**

```js
const fs = require('fs')
fs.stat('./test.txt', (err, info) => {
    console.log(err)
    console.log(info)
})
fs.stat('./test2', (err, info) => {
    console.log(err)
    console.log(info)
})
```

- **方式三：promise方式**

```js
const fs = require('fs')
fs.promise.stat('./test.txt').then(res => {
    console.log(res)
}).catch(err => {
    console.log(err)
})
fs.promise.stat('./test2').then(res => {
    console.log(res)
}).catch(err => {
    console.log(err)
})
```
