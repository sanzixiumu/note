# 初始 css

css:层叠样式表，可以使用 CSS 来更改内容的字体、颜色、大小、间距，将内容分为多列，或者添加动画及其他的装饰效果。

## 引入样式

1. 内部样式

```css
<style>
h1 {
    color: red;
}
</style>
```

2. 外部样式

```css
<link rel="./style.css">
```

3. 标签样式

```css
<h1 style="color: blue;">h1</h1>
```

## 优先级

- 标签样式

  - 优先级：最高
  - 缺点：只能作用于该标签

- 外部样式

  - 优先级：高
  - 缺点：需要引入

- 内部样式
  - 优先级：低
  - 缺点：只能作用于该页面

> 注意：同时拥有内部样式和外部样式时，后面的会覆盖前面的
