# 列表元素

用来展示列表的标签

## 有序列表

告诉浏览器和搜索引擎，这个部分是有序列表

- ol li

```html
<h2>把大象装冰箱，总共分几步</h2>
<ol>
  <li>把冰箱门打开</li>
  <li>把大象装进去</li>
  <li>把冰箱门关上</li>
</ol>
```

## 无序列表

告诉浏览器和搜索引擎，这个部分是无序列表

- ul li

```html
<h2>动物园开会，有哪个动物没有去</h2>
<ol>
  <li>大象</li>
  <li>老鼠</li>
  <li>狮子</li>
</ol>
```

## 定义列表

- dl dt dd

```html
<dl>html中有哪些技术
    <dt>
        html
    <dd>html5</dd>
    <dd>css</dd>
    <dd>JavaScript</dd>
    </dt>
</dl>
```
